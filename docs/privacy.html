🔒 隱私權原則

Privacy Policy  ·  最後更新：2026 年 3 月

📢 摘要：本擴充套件不收集、不儲存、不傳送任何個人資料。 所有操作皆在您的瀏覽器本地執行，僅在您主動點擊按鈕時才向第三方 API 發送 PoB 代碼。

📌 1. 適用範圍

本隱私權原則適用於「PoE Ninja to Chronicles PoB Sharer」瀏覽器擴充套件（以下簡稱「本套件」），說明本套件如何處理使用者資料。

📊 2. 資料收集

本套件不收集任何個人識別資訊，包含但不限於：

  • 姓名、電子郵件或帳號資訊
  • IP 位址或裝置識別碼
  • 瀏覽紀錄或行為分析資料
  • Cookie 或追蹤識別碼

⚙️ 3. 套件運作方式

本套件僅在以下情況下執行動作：

  • 在 poe.ninja 角色頁面注入「分享中文 PoB」按鈕（純介面操作）
  • 當您主動點擊按鈕後，讀取該頁面公開的 PoB 代碼
  • 將 PoB 代碼透過 HTTPS 上傳至 poe2db.tw（編年史）公開 API
  • 將回傳的分享連結自動複製至剪貼簿

本套件不會在背景自動執行任何網路請求，也不會存取任何與 PoE 無關的頁面。

🌐 4. 第三方服務

使用者主動觸發分享時，PoB 代碼將傳送至以下第三方服務：

  • poe2db.tw（編年史） — 用於儲存 PoB 代碼並產生分享連結

PoB 代碼為 Path of Exile 公開角色配裝資料，不含任何個人識別資訊。請參閱第三方服務的隱私權政策以了解其資料處理方式。

🔑 5. 所需權限說明

  • activeTab — 僅存取您目前瀏覽的 poe.ninja 頁面，讀取公開的 PoB 代碼
  • clipboardWrite — 將產生的分享連結複製至剪貼簿
  • host_permissions（poe.ninja、poe2db.tw） — 注入介面按鈕及呼叫分享 API

🔄 6. 本原則的變更

若本套件功能或資料處理方式有重大變更，將更新本頁面並修改「最後更新」日期。建議定期查閱以掌握最新資訊。
